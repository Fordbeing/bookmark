## 🚀 快速开始步骤

### 1. 环境准备
```bash
# 确保安装了Node.js 14+
node --version
npm --version
```

### 2. 项目安装
```bash
# 进入项目目录
cd bookmark-frontend

# 安装所有依赖
npm install
```

### 3. 启动开发服务器
```bash
# 启动开发服务器
npm run dev

# 应用将在 http://localhost:5174 打开
```

### 4. 生产构建
```bash
# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
