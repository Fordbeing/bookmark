## 📊 技术实现细节

### 组件架构
```
App.vue (主控制器)
├── Sidebar.vue (侧边栏 + 折叠功能)
├── Navbar.vue (顶部导航 + 设置按钮)
├── AddBookmarkPage.vue (手工添加页面)
├── EditBookmarkPage.vue (编辑页面)
├── AddBookmarkModal.vue (快速添加弹窗)
├── LoginPage.vue (登录页面)
└── SettingsPage.vue (完整设置对话框)
```

### 状态管理
- `currentPage`: 当前页面状态 ('main'|'add'|'edit')
- `editingBookmark`: 正在编辑的书签对象
- `isSettingsVisible`: 设置对话框显示状态
- `localStorage`: 用户偏好设置持久化

### 事件流
```
Navbar (设置按钮)
  └─> @toggle-settings
      └─> App.vue (isSettingsVisible = true)
          └─> SettingsPage (@update-settings)
              └─> handleSettingsUpdate()
                  └─> localStorage 保存配置
```

## 🚀 使用指南

#### 1. 手工添加书签
```javascript
// 点击导航栏中的"手工添加"按钮
// 或点击底部悬浮按钮
currentPage.value = 'add'
```

#### 2. 编辑书签
```javascript

editBookmark(item)
// 页面自动导航到编辑页面
```

#### 3. 打开设置
```javascript
// 点击顶部导航栏的"⚙️"按钮
// 或点击侧边栏的"设置"按钮
```

