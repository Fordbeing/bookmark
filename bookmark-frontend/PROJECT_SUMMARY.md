## 🚀 快速开始

### 安装和运行
```bash
# 进入项目目录
cd bookmark-frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问应用
# 打开浏览器访问 http://localhost:5174
```

### 主要功能演示
1. **查看预置书签**：页面加载时自动显示6个示例书签
2. **快速添加书签**：在顶部输入链接，点击"立即添加"或按Enter
3. **详细添加书签**：点击右下角"+"按钮，填写完整信息
4. **删除书签**：点击书签卡片上的"🗑️"按钮
5. **打开书签**：直接点击书签卡片打开链接
6. **查看设置**：点击侧边栏"⚙️ 设置"按钮

## 🎓 学习价值

这个项目展示了以下最佳实践：

### Vue 3 Composition API
```javascript
// 使用ref和computed
const count = ref(0);
const doubled = computed(() => count.value * 2);

// 使用watch
watch(count, (newVal) => {
  console.log('Count changed to:', newVal);
});
```

### Element Plus集成
```vue
<!-- Dialog组件 -->
<el-dialog v-model="visible" title="标题" />

<!-- Form组件 -->
<el-form :model="form" label-width="80px">
  <el-form-item label="标题"><el-input /></el-form-item>
</el-form>
```

### Tailwind CSS应用
```vue
<!-- 响应式设计 -->
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">

<!-- 悬停效果 -->
<div class="hover:shadow-lg hover:-translate-y-2 transition-all duration-300">

<!-- 渐变背景 -->
<div class="bg-gradient-to-b from-blue-600 to-blue-800">
```

### 组件通信
```vue
<!-- Props向下传递 -->
<Child :title="title" :description="description" />

<!-- Emits向上发送 -->
<button @click="$emit('close')">关闭</button>

<!-- v-model双向绑定 -->
<Dialog v-model:visible="isVisible" />
```

## 📞 开发者指南

### 添加新功能的步骤
1. 在对应组件中添加新的state
2. 实现相关的方法或计算属性
3. 在模板中添加对应的UI
4. 添加相关的样式
5. 在文档中记录更改

### 修改样式的方法
- 使用Tailwind CSS类名
