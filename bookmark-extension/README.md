# BookUtil 浏览器扩展

书签管理系统的浏览器扩展，支持 Chrome 和 Edge 浏览器。

## 功能特性

- ✨ **一键保存** - 快速保存当前页面为书签
- 📁 **分类管理** - 选择或创建书签分类
- 🏷️ **标签支持** - 添加多个标签便于组织
- ⭐ **收藏标记** - 标记重要书签
- 🔍 **快速搜索** - 在扩展中搜索已保存的书签
- ⌨️ **键盘快捷键** - `Ctrl+Shift+B` 快速保存当前页面
- 🖱️ **右键菜单** - 右键保存链接或页面

## 安装方法

### Chrome / Edge 浏览器

1. 打开浏览器扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`

2. 开启右上角的 **「开发者模式」**

3. 点击 **「加载已解压的扩展程序」**

4. 选择 `bookmark-extension` 目录

5. 扩展安装完成！点击浏览器工具栏中的 BookUtil 图标即可使用

## 使用方法

### 登录
首次使用需要登录您的 BookUtil 账户：
1. 点击扩展图标打开弹窗
2. 输入用户名和密码
3. 点击登录

### 保存书签
**方式一：使用弹窗**
1. 访问想要保存的网页
2. 点击扩展图标
3. 编辑标题、描述、选择分类
4. 点击「保存书签」

**方式二：使用快捷键**
- 按 `Ctrl+Shift+B`（Mac: `Cmd+Shift+B`）直接保存当前页面

**方式三：使用右键菜单**
- 在页面空白处右键 → 选择「保存到 BookUtil」
- 在链接上右键 → 选择「保存链接到 BookUtil」

### 搜索书签
1. 点击扩展图标
2. 切换到「我的书签」标签
3. 在搜索框输入关键词

## 故障排除

### 登录失败
- 检查网络连接
- 确认用户名密码正确
- 确认后端服务正在运行

### 保存书签失败
- 检查是否已登录
- 某些特殊页面（如 chrome:// 开头的页面）无法保存

### 快捷键不生效
- 检查快捷键是否与其他扩展冲突
- 在 `chrome://extensions/shortcuts` 中可以自定义快捷键
